# Smoke test: Mock workflow

1. Start the app:
   - `npm run dev`
2. Sign in with Google (must be configured).
3. Go to `/app`.
4. Upload a PDF mock or paste mock text, then click **Analyze**.
5. Confirm the insights, next best actions, plan, and progress tracker render.
6. Change a task status to **Done** or **Skipped**, add a note, and refresh the page.
7. Verify the task status persists and the progress tracker updates.
